## ✨ Features & Visual Highlights

This portfolio is crafted to look premium, immersive, and highly interactive. Key highlights include:

*   **🧪 3D Glass Profile Card:** A frosted glass card (`backdrop-filter: blur(24px)`) featuring a sleek mouse-follow 3D tilt effect, active scan line, cyan neon border accents, and floating tag chips.
*   **🌌 Dynamic Aurora Canvas:** A native HTML5 `<canvas>` background rendering custom-positioned floating radial gradient blobs and twinkling star particles with real-time mouse proximity glow.
*   **💫 Adaptive Custom Cursor:** A dual-element custom cursor containing a solid core and a lagging outer ring that grows and glows when hovering over interactive elements.
*   **📈 Smooth Reveal Animations:** Fully responsive scroll-reveal animations implemented natively using JavaScript's `IntersectionObserver` API.
*   **📊 Animated Skill Bars:** Skills loaded with dynamic progress fills that activate only when scrolled into view.

## 🚀 Running Locally

Because this project is built entirely with vanilla frontend technologies (HTML5, CSS3, and JavaScript), it does not require any compile or build steps.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Supriya57-komla/MyPortfolio.git
    ```
2.  **Navigate into the directory:**
    ```bash
    cd MyPortfolio
    ```
3.  **Open in your browser:**
    Simply double-click the `index.html` file or run it via a local development server (e.g., Live Server extension in VS Code).
